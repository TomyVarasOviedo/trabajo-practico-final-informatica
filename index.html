¿Que es la CPU?

Unidad Central de Procesamiento

Unidad de Control

  • Controlar y supervisar todo el sistema de cómputos en base a un programa almacenado en UMC.
  • Desarrollar las operaciones aritméticas y lógicas que sirven para procesar los datos. Controlando las secuencias de ejecución de las Instrucciones.
  • Controlar el envío/recepción de datos desde las unidades de I/O a la UMC, y viceversa, para ello usa el DMA (Direct Memory Access)

Unidad Arimético-Lógica

Es el módulo que realiza las Operaciones Matemáticas y Lógicas. Tales como suma, resta, multiplicacion, division y lógicas como comparaciones atravez de los datos brindados por la unidad de control codificados de forma binaria.

Unidad de Memoria

Este módulo está compuesto por un mapeo de ROM y RAM (Memoria Volátil). Guarda los programas que se van a ejecutar. Almacena Instrucciones, datos, y otras direcciones. Se constituye en registros de UMC, compuestos por celdas, y son de longitud fija.
